1.安装node

	brew install node

2.安装RN命令行工具

	npm install -g react-native-cli

3.测试安装

	react-native init AwesomeProject
	cd AwesomeProject
	react-native run-ios

运行成功，使用编辑器打开index.ios.js并随便改上几行。
在iOS Emulator中按下⌘-R就可以刷新APP并看到你的最新修改！
